
## Gitlab

- Activation du Container Registry
- Création d'une image docker alpine contenant le certificat interne + le binaire bao
- Création d'une image docker python + uv avec certificat interne pour lancer les scripts Python en CI/CD.
- Création d'un catalogue CI/CD permettant de standardiser et unifier les jobs répétitifs
- Mise en place d'un job automatisé de query de secrets openbao via JWT.
### Image-Factory/Create VM Template

- Création du script python permettant de faire la première partie du travail de l'image factory. Voir ADR [[Image Factory Générique]] (Linux only.)

## OpenBao

- Création d'une auth de type JWT pour Gitlab (voir [[OpenBao Vaults]]), configuration de la discovery URL de cette dernière.
- Mise en place de policy dynamique sur le vault kv CI.
- Création de la policy et du rôle associés a gitlab.
- Création d'une 2e policy permettant à la CI d'accéder à l'infrastructure en Read-Only.
